# Linux
## 필수 명령어
- ls -> 현재 위치의 파일목록 조회
```
pwd -> 현재 자신의 위치
```
- cd (change directory) -> 디렉터리 이동
```
cd / = 최상위 폴더로 이동
cd ~ = /최상위폴더/username 
cd .. = 현재 위치의 상위 폴더로 이동
cd - = 자신이 있었던 전 폴더로 이동
```

- mkdir(make directory) -> 디렉터리 제작
```
sudo = 권한대행
```

- cp(copy pile) -> 파일 복사 ( cp (복사할파일) (복사될파일) )

- mv(move file) -> 파일 이동 copy와 다르게 원본파일 이동(삭제))
- rm(remove) -> 파일 삭제
```
rm (삭제할 폴더) -rf
```
- cat -> 파일 내용을 화면에 출력, 리다이렉션 (기호 : '>')을 사용하여 새로운 파일 생성
